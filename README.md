# 🛒 Walmart Sales Analysis | SQL Project

## 📌 Overview
This project analyzes Walmart sales data using SQL to uncover insights into **sales performance, customer behavior, and product trends**.  
It includes data cleaning, feature engineering, and exploratory analysis using structured queries.

---

## 🚀 Key Features
- 🧹 Data cleaning and preprocessing
- ➕ Feature engineering:
  - Time of day (Morning, Afternoon, Evening)
  - Day name and month name extraction
- 📊 Sales and revenue analysis
- 📦 Product line performance analysis
- 👥 Customer behavior insights
- ⭐ Rating and satisfaction analysis
- ⏱️ Time-based sales trends

---

## 🛠️ Tools & Technologies
- SQL (MySQL)
- Data Analysis
- Relational Database Design

---

## 📂 Repository Contents
- `WalmartSalesData.csv` → Raw dataset  
- `SQL_queries.sql` → Complete SQL analysis queries  

---

## 🔍 Key Analysis Performed

### 🏙️ General Analysis
- Unique cities and branches  
- Branch-wise operations  

### 📦 Product Analysis
- Top-selling product lines  
- Revenue by product category  
- Product performance classification (Good/Bad)  

### 👥 Customer Analysis
- Customer types (Member vs Normal)  
- Payment method usage  
- Gender distribution  
- Ratings analysis  

### 📅 Time-Based Analysis
- Sales trends by day and month  
- Peak sales time (Morning/Afternoon/Evening)  
- Weekly performance  

### 💰 Sales Insights
- Revenue by customer type  
- Tax/VAT analysis  
- City-wise revenue comparison

1. Import the dataset (`WalmartSalesData.csv`) into MySQL
